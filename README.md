🔔 Angular Reminder SystemA high-performance, reactive web application built to schedule and trigger system-level notifications. This project demonstrates a deep understanding of Angular's modern architecture and the Web Notifications API.🚀 Key FeaturesSystem Notifications: Leverages the browser's hardware API to send alerts directly to the OS tray.Reactive Data Handling: Uses Angular FormBuilder for robust user input validation.Modern Control Flow: Implements the new @for and @empty syntax for efficient DOM rendering.Service-Oriented Architecture: Decouples notification logic (Service) from the user interface (Component).🛠️ Technical StackFramework: Angular 17+ (Standalone Components)Language: TypeScriptStyling: CSS3 (Flexbox/Grid)Logic: JavaScript Event Loop (setTimeout) & Web Notifications API📂 Project StructureReminderService: Handles browser permissions and calculates the $delay$ for setTimeout.AppComponent: Manages the form state and handles the synchronization between the UI and the data service.Reminder Interface: Ensures strict typing for reminder objects (ID, Subject, Content, DateTime).📥 How to Run LocallyClone the repo:Bashgit clone https://github.com/Preedpn/angular-reminder-system.git
